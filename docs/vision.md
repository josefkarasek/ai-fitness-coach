# Vision

## Goal

Create an AI strength coach that understands years of training history
and produces personalized long-term training plans.

The app should feel like working with a human coach.

The core product logic should live in Go so that mobile clients stay
thin and consistent.

## MVP

-   Import TrainHeroic exports
-   Build athlete profile
-   Implement Go training engine
-   Generate 12-week training block
-   Generate a Coaching Book
-   Explain every exercise ("Why is this here?")
-   Expose engine capabilities over REST API
-   Offline workout logging
-   Flutter mobile client
-   End-of-block AI review

## Non-goals

-   Nutrition
-   Social features
-   Wearables
-   Continuous AI chat
